The AI Pipeline

Follow a query from prompt to verified response.

User Query

"When is the CS101 exam, and how do I register?"

Agent Node

Analyzes user intent, verifies campus scope, and rewrites the query using chat memory.

Retriever

Generates context embeddings via Gemini and retrieves highly relevant chunks from Qdrant data store.

Answer Node

LLM synthesizes retrieved context and applies strict guardrails to return an academic, hallucination-free response.
